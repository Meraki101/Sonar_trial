Minimal Form Example Basic Usage:

1) Create a list called "minimal" with the following field:
	fieldName: StratusFormsData
	fieldType: multi-line plain text
2) Upload the examples to a SiteAssets library with the same folder structre
3) Create a page in SharePoint
4) Add a Content Editor Web Part to the page
5) Link the Content Editor Web Part to SiteAssets/RichText/RichText.html
6) Ta Da!
